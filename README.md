# Spam-call-Detection-
Project Description

This project is a machine learning–based spam call classification system developed using Python and scikit-learn. The system analyzes call behavior data to identify whether a call is spam or legitimate.

Technologies Used
Python
scikit-learn
Pandas
NumPy
Main Features
Spam call classification using Machine Learning
Implemented Logistic Regression and Random Forest algorithms
Data preprocessing and feature extraction
Model evaluation using accuracy metrics
Hyperparameter tuning and cross-validation for better performance
Accuracy
Achieved 92% accuracy in spam call detection.
How to Run
Install required libraries:
pip install pandas numpy scikit-learn
Run the project:
python spam_call_detection.py
